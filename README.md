# it_expert_linux_device_driver

# 라즈베리 파이2 용 소스
## 설명
 라즈베리파이2에서 실습할 수 있는 소스를 제작 배포

## 라즈베리 파이 2 패치지 설치
```sh
$ sudo apt-get install build-essential git vim raspberrypi-kernel-headers
$ mkdir ~/git
$ cd ~/git
$ git clone https://github.com/iamroot12CD/it_expert_linux_device_driver
```
